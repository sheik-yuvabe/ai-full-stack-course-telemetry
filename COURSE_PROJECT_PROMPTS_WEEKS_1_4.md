# Course Project Prompts: Weeks 1-4

This file gives one ready-to-use prompt for each main project from Weeks 1 to 4.

You can copy the prompt for the current project and paste it into Codex.

If you are doing an alternative project, you can use the same prompt structure and replace the project name and app idea.

## Week 1

### Week 1 Day 1: Daily Pulse Board

```text
I'm in Week 1 focusing on creating Daily Pulse Board.

I want to build a small React app which includes a few selectable options.
When a user clicks on an option, the screen should change according to the mood, energy level, or focus level.
```

### Week 1 Day 2: Playlist Duel

```text
I'm in Week 1 focusing on creating Playlist Duel.

I want to build a small React app where users can vote between songs or albums and the app shows the current leader.
```

### Week 1 Day 3: Workshop RSVP + Ticket Preview

```text
I'm in Week 1 focusing on creating Workshop RSVP + Ticket Preview.

I want to build a small React app with a signup form and a live preview of the attendee pass or ticket.
When the user types into the form, the preview should update on the screen.
```

### Week 1 Days 4-5: Movie Night Planner

```text
I'm in Week 1 focusing on creating Movie Night Planner.

I want to build a small React app where users can browse a mock movie list, search by title, filter by genre, and save a shortlist.
```

## Week 2

### Week 2 Days 6-7: Reading Queue

```text
I'm in Week 2 focusing on creating Reading Queue.

I want to build a React app where users can add books or articles, update status, edit notes, and remove items.
```

### Week 2 Day 8: Weekend Budget Splitter

```text
I'm in Week 2 focusing on creating Weekend Budget Splitter.

I want to build a React app where users can enter shared expenses and see totals and per-person split.
```

### Week 2 Days 9-10: Recipe Box

```text
I'm in Week 2 focusing on creating Recipe Box.

I want to build a React app where users can add recipes, tag them, filter them, save favorites, and keep the data after refresh.
```

## Week 3

### Week 3 Days 11-15: City Weekend Planner

```text
I'm in Week 3 focusing on creating City Weekend Planner.

I want to build a larger React app where users can browse a local dataset of cafes, parks, events, exhibitions, and food spots, filter the options, build a weekend plan, save items, remove items, and keep data after refresh.
```

## Week 4

### Week 4 Days 16-20: Rental Scout

```text
I'm in Week 4 focusing on creating Rental Scout.

I want to build a frontend-only React app with browse page, filters, favorites or saved items, detail pages, and an inquiry form.
```

## How to use alternative projects

If you are doing an alternative project, keep the same prompt structure and replace:

- the project name
- the app idea sentence

Examples:

- replace `Daily Pulse Board` with `Coffee Preference Picker`
- replace `Playlist Duel` with `Trailer Showdown`
- replace `Reading Queue` with `Watchlist Manager`
- replace `Rental Scout` with `Event Discovery App`
